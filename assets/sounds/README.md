# assets/sounds/

Place four `.mp3` files here to enable sound effects in PräFix.

| File | Description | Length |
|---|---|---|
| `correct.mp3` | Short bright chime — played on correct answer | ~0.5 s |
| `wrong.mp3` | Soft low buzz — played on wrong answer | ~0.5 s |
| `complete.mp3` | Celebratory fanfare — played on level complete | ~2 s |
| `unlock.mp3` | Ascending ding — played when a new level unlocks | ~1 s |

## Free sources

- **freesound.org** — search: "correct chime CC0", "error buzz CC0"
- **mixkit.co/free-sound-effects/** → Game → Notifications
- **jfxr.frozenfractal.com** — browser-based 8-bit/retro sound generator (free)
- **sfxr.me** — classic game sound generator, exports .wav → convert to .mp3

## After adding the files

Follow the 3-step instructions at the top of `hooks/useSound.ts`.
